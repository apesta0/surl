# 🤝 Code of Conduct

## Our Pledge

We aim to make this project a welcoming and friendly space for everyone, regardless of experience level, background, or identity.

---

## Expected Behavior

- Be respectful and kind
- Use welcoming and inclusive language
- Accept constructive feedback gracefully
- Focus on what is best for the project

---

## Unacceptable Behavior

- Harassment or discrimination of any kind
- Offensive or harmful comments
- Personal attacks
- Spamming or disruptive behavior

---

## Scope

This Code of Conduct applies to:
- GitHub repository interactions
- Issues and pull requests
- Discussions about the project

---

## Enforcement

Violations may result in:
- Warning
- Temporary restriction
- Permanent removal from project participation

---

## Contact

If you experience or witness unacceptable behavior, please report it via GitHub issues or repository maintainer contact.
